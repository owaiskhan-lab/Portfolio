# Java

This folder contains Java-related examples and starter files for the Portfolio repository.

## Structure
- java/README.md - this file
- java/src/HelloWorld.java - simple example Java program

## How to compile and run
1. Open a terminal in the repository root.
2. Compile: `javac java/src/HelloWorld.java`
3. Run: `java -cp java/src HelloWorld`
